# 杀戮尖塔2 地图绘制工具 - 使用说明

## 环境准备

如果依赖未安装，请先运行：
```
pip install -r requirements.txt
```

## 运行程序

```
python main.py
```

## 使用步骤

### 1. 导入图片
- 点击「📂 选择图片」按钮（图片的分辨率不要尽量太大，不然路径点过多会导致速度变慢）
- 选择要绘制的图片（支持 PNG、JPG、BMP、GIF 等格式）
- 程序会自动进行边缘检测并显示效果

### 2. 调整边缘检测参数
- 使用「低」「高」滑块调整 Canny 边缘检测阈值
- 阈值越低，检测到的线条越多
- 可以在「边缘检测效果」预览区域查看效果

### 3. 框选图像区域
- 点击「📐 框选图像区域」
- 在右侧「边缘检测效果」图像上按住鼠标左键拖动
- 选择要绘制的图案区域

### 4. 框选游戏绘制区域
- 点击「🎮 框选绘制区域」
- 3秒后程序会最小化，切换到杀戮尖塔2游戏窗口
- 点击游戏地图的左上角，然后拖动到右下角选择绘制区域
- 程序会自动恢复

### 5. 开始绘制
- 点击「🖌️ 开始绘制」按钮，点击游戏地图上的画笔
- 5秒倒计时后程序会最小化
- 快速切换到游戏窗口，鼠标将自动绘制图案，鼠标右键可停止绘画

## 参数说明

- **边缘检测阈值**：控制线条检测的灵敏度
  - 低阈值：50（默认）
  - 高阈值：150（默认）
- **绘制速度**：控制绘制快慢
  - 越向右越快,但是可能会损失精度
